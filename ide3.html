Visual Studio Code

L'argent n'a pas d'codeur?

En juste quelques années, et quelques millions de dollars, VSCode est devenu un editeur de choix pour beaucoup de développeurs (source).
Outre le nom de Microsoft, l'éditeur a de vrais avantages, comme son marketplace, ses choix de configurations et son intégration native de Github. Sans doute le rachat de Github n'y est pas pour rien.

Rachat par Microsoft de Github

Résumons!

Avantages:

  • la customisation des thèmes ( et les thèmes eux-mêmes)
  • Basse consommation en RAM
  • Facilité d'accès et d'utilisation
  • intégration d'un terminal
  • Intégration de Git
  • Possibilité d'utiliser sans limite de temps
  • le support financier de Microsoft, il y a pire comme situation
  • Tous les plugins de l'univers

Inconvénients:

  • Beaucoup, beaucoup de commandes disponibles
  • Certains langages de programmation pas encore disponibles
  • Un terminal qui peut être capricieux
  • des raccourcis clavier demandant certaine dextérité
  • L'abondance peut faire qu'on s'y perde pendant de longs moments
